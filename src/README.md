#  Billing System (Desktop Application)

A professional desktop billing system built for MS Traders using:
- Python 3.13
- Tkinter (UI)
- SQLite (Local database)
- ReportLab (PDF invoice generator)
- PyInstaller (EXE build)

This system supports:
- Customer management
- Bill generation
- Multiple item entries
- Professional invoice PDF with logo + signature
- Daily & Monthly reporting
- Multi-selection invoicing
- Customer history panel
- Silver-Grey corporate UI theme
